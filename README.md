# PLST Derivation

This repository contains the clean first-principles three-generation derivation manuscript.

## Contents

- `paper/main.tex` - LaTeX source for the derivation.
- `paper/main.pdf` - compiled PDF.

The core claim is that the three Standard Model generations arise from the two-center projection divisor
`R = p_+ + p_-` on `CP^1`, giving the generation bundle `O(R) = O(2)` and
`dim H^0(CP^1, O(2)) = 3`.
